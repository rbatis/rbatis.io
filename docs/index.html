RBatis

Compile-time ORM for Rust

⚡

High Performance

Zero-cost dynamic SQL compiled at build time

🛡️

Type Safe

100% safe Rust, compile-time query verification

🔌

Driver Abstraction

Pluggable database drivers via rbdc trait system

# Cargo.toml
$ cargo add rbatis

# Quick start
$ let rb = rbatis::RBatis::new();
$ rb.link("mysql://user:pass@localhost/db").await;
$ let users: Vec<User> = rb.fetch().await;
