Why RBatis?

A modern, high-performance ORM for Rust with compile-time safety

⚡

Compile-time Dynamic SQL

Zero-cost dynamic SQL powered by proc-macros at compile time. Uses Cow to minimize cloning — no ONGL engine needed.

🔄

MyBatis3 Compatible

Familiar MyBatis3 syntax support. Easily migrate existing Java projects to Rust with minimal changes.

🛡️

100% Safe Rust

Enforces #![forbid(unsafe_code)] — compile-time query verification, zero undefined behavior.

🚀

Async & High Performance

Built on Tokio async runtime. No GC, no runtime overhead — pure zero-cost abstractions.

🔌

Pluggable Drivers

JDBC-like rbdc trait system. Switch databases by changing a single line of Cargo.toml.

🧩

Rich Plugin System

Interceptors, auto table-sync, snowflake & ObjectId generators, pagination, and more.

Dynamic SQL

Two powerful dynamic SQL languages — write complex queries with ease

html_sql

MyBatis-compatible XML syntax with <if>, <foreach>, <where>, <trim>, and <choose> support.

<select id="select_by_condition">
`select * from biz_activity`
<where>
<if test="name != ''">
` and name like #{name}`
</if>
</where>
</select>
py_sql

Python-like syntax for dynamic SQL — concise, readable, and expressive.

# py_sql example
`select * from user`
where :
if name != '':
` and name=#{name}`
if delete_flag != 0:
` and delete_flag = 0`
